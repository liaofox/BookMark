🖥️ 私人网址收藏夹

一个基于Flask的私人网址收藏管理系统

注意： 这是一个服务器端应用，无法直接在GitHub Pages上运行。 GitHub Pages只支持静态网站，而本应用需要Python Flask服务器。

📦 本地运行方法

  1. 克隆仓库到本地：git clone https://github.com/你的用户名/你的仓库名.git
  2. 安装依赖：pip install -r requirements.txt
  3. 启动应用：python run.py
  4. 访问：http://localhost:5000

☁️ 云端部署选项

  • Heroku：免费的云平台，支持Python应用
  • PythonAnywhere：专门针对Python应用的托管服务
  • Railway：现代化的应用部署平台
  • Render：另一个优秀的云部署平台

📚 项目特性

  • 用户注册和登录系统
  • 安全的密码哈希存储
  • 网址收藏管理功能
  • SQLite数据库存储
  • 响应式网页设计
